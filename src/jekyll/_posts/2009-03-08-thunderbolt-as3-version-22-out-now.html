--- layout: post title: ThunderBolt AS3 version 2.2 out now! tags: - AIR - Flash - Flex - Open Source - ThunderBolt status: publish type: post published: true meta: _edit_last: "2" --- ThunderBolt AS3, which is a lightweight logger extension for Flex 3-4, AIR and Flash 9-10 applications, has been updated to version 2.2. Check out the new features:

New features

  1. Logging of package names + class names + code line numbers (Flash Debug Player required).

  2. ThunderBoltTarget for using Flex Logging API improved.
  3. SWC updated for using Flex 4 (Gumbo)
  4. ThunderBolt AS3 Console uses a certificate of Thawte sponsored by Adobe.

  5. ThunderBolt AS3 Console is updated using latest version of Adobe AIR v.1.5.1
  6. Issue 10 fixed: "Command + Q not working in OS X"
  7. All examples and documentation has been updated as well.
Happy Logging ;) -Jens
