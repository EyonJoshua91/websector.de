UPDATE [10/31/10]: Example and its full source has been updated using latest commit of Swiz 1.0 RC2.

Cafe Townsend is a well known application to demonstrate any MVC framework for using Flex or ActionScript. There are already ports of Cairngorm, Mate, PureMVC, Spring ActionScript, RobotLegs and now Swiz. In cooperation with "my partner in crime" Sönke Rohde we have built a Swiz port including unit tests.
